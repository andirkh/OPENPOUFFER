#1. Framework DIY OPENPOUFFER 
oleh @andirkh & @pouffer.io (instagram.com/pouffer.io) merupakan gambar vector lelaki dan perempuan yg bebas kalian edit sendiri menjadi karya lain sesuai kreativitas kalian. Berguna untuk tugas sekolah/kuliah atau untuk produk komersil seperti kaos, game visual novel, karakter buku cerita, packaging, maskot produk dan peluang tak terbatas lainnya.

Seperti lisensi BY-SA yg lain, Kalian Bebas untuk :
"Share — copy and redistribute the material in any medium or format
Adapt — remix, transform, and build upon the material
for any purpose, even commercially."

Format : .svg
Tools yg digunakan :
- Komputer Windows, OSX, Linux
- Software Inkscape 0.91

#2. OpenPouffer - Budoyo
merupakan project free open source, aset 2d tentang kebudayaan Indonesia. Berisi karakter dengan baju adat 34 Provinsi Indonesia (masih on going). Asset 2 dimensi disini bisa kalian gunakan juga untuk tugas sekolah / kuliah, dijadikan produk turunan lainnya seperti game dress-up, packaging product, karakter di software apps kalian, dan lain-lain. Boleh juga untuk kepentingan komersil, namun jgn lupa menulis sumbernya :)
