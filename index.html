Perkenalkan

OpenPouffer menyediakan asset vektor 2D gratis dan open source tentang kebudayaan 32 provinsi Indonesia yg berguna untuk :

  • Tugas Kuliah / Sekolah
  • Membuat produk kreatif seperti Game, Visual Novel, Kemasan Produk dll
  • Kepentingan pribadi maupun komersial

Project ini belum sepenuhnya selesai, baru sebagian terbuat namun sudah bisa digunakan. Check terus Github page-nya disini dan jika ingin mendownload klik disini

"Update 2 Oktober 2016 : Dua belas provinsi sudah selesai, tinggal 20 provinsi."
