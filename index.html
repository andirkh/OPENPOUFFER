Tentang

Pouffer menyediakan asset vektor 2D gratis dan open source tentang kebudayaan 34 provinsi Indonesia yg berguna untuk Tugas Kuliah / Sekolah, Membuat produk kreatif seperti Game, Visual Novel, Kemasan Produk dll, Kepentingan pribadi maupun komersial

Project ini belum sepenuhnya selesai, baru sebagian terbuat namun sudah bisa digunakan. Check terus Github-nya disini dan jika ingin berkontribusi, jangan ragu untuk hubungi saya lewat Telegram di tombol di bawah.
